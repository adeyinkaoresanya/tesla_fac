# Tesla
Tesla is a Python library for building a simple model of Tesla cars

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install tesla.

```bash
pip install tesla
```

## Usage

```python
import tesla

# returns the colour of the car
tesla.color('red')

# returns the number of seats
tesla.seats_count()

# unlocks the car
tesla.unlock()
```
